# 課題８について

課題８のような条件分岐を実現するには、機能カテゴリにある `switch` ノードを使う。

# スマホでの確認

`SSID: IoT` に接続して、スマホから Node-RED サーバーにアクセスして、スマホから PLC を操作してみよう。

接続するアドレスは 
> `http://192.168.x.x:1880/ui`

ファイアウォールがブロックしているケースもあるのでそのときは呼んでください。

# FA システムセットアップ

第１～第６ステーションが（いつもの FA の授業と同じように）稼働するようにシステムをセットアップしてください。

# 各ステーションの制御

第１～第６ステーションの各ステーションの M10(スタート) / M3(サイクル停止) のボタンを作成し、Node-RED からすべてのステーションが操作できることを確認してください。

# Y10 「稼働中」サインの反映

dashboard 画面から各ステーションの Y10 (コンベアモーター) のモニタリングができるようにしてください。
